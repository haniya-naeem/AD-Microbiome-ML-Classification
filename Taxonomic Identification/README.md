Taxonomic Identification and Preprocessing

This notebook shows the amount of gut bacteria compared to each other based on levels of Alzheimer’s Disease (AD), Mild Cognitive Impairment (MCI), and healthy samples. The process includes uploading the data, cleaning the data, filtering out low-frequency data, looking at average values between groups, and creating a PCA. Additionally, this will highlight which bacteria-systems have the largest amount of variability between groups, and create a standard set of systems to be used in modeling and visualizing in the future.
